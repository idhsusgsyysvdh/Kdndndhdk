## 🚀 Panduan Instalasi (VPS / Ubuntu 24)

Kami telah menyederhanakan proses instalasi dengan script otomatis.

### 1. Update Server
Pastikan server Anda sudah terupdate.
```bash
sudo apt update && sudo apt upgrade -y
```

### 2. Clone Repository
Ambil source code terbaru dari GitHub SherifNet.
```bash
git clone https://github.com/sherif1day-ux/userbot.git
cd userbot
```

### 3. Jalankan Setup Otomatis
Script ini akan menginstall Python 3.12, ffmpeg, dan semua library yang dibutuhkan secara otomatis.
```bash
bash setup.sh
```

### 4. Konfigurasi Bot
Jalankan script konfigurasi untuk mengisi data bot Anda secara interaktif.
```bash
bash set.sh
```
*Script ini akan meminta `API_ID`, `API_HASH`, `BOT_TOKEN`, `MONGO_URL`, dll dan otomatis membuat file `.env`.*

### 5. Jalankan Userbot
Mulai bot Anda dengan perintah berikut:
```bash
bash start.sh
```

## 🛠️ Variabel Konfigurasi (.env)

Berikut adalah variabel penting yang perlu Anda isi di file `.env`:

| Variabel | Deskripsi | Cara Mendapatkan |
| :--- | :--- | :--- |
| `API_ID` | ID API Telegram Anda | [my.telegram.org](https://my.telegram.org) |
| `API_HASH` | Hash API Telegram Anda | [my.telegram.org](https://my.telegram.org) |
| `BOT_TOKEN` | Token Bot Utama | [@BotFather](https://t.me/BotFather) |
| `OWNER_ID` | ID Akun Telegram Anda | [@userinfobot](https://t.me/userinfobot) |
| `MONGO_URL` | URL Koneksi Database MongoDB | [MongoDB Atlas](https://www.mongodb.com/) |
| `RMBG_API` | API Key Remove.bg (Opsional) | [remove.bg](https://www.remove.bg/) |
| `LOGS_MAKER_UBOT` | ID Channel/Group Log (Opsional) | Buat grup & ambil ID-nya |
